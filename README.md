🌍 GeoRawi – Interactive Documentary Map

📖 Overview

GeoRawi is an interactive Arabic storytelling map that transforms real-world locations into cinematic historical experiences.

Instead of traditional navigation, users explore nearby landmarks and receive documentary-style stories about each place, creating an immersive journey through history and geography.

⸻

🚀 Live Demo

👉 [https://geo-rawi.replit.app]

⸻

🗺️ Features

* Interactive map powered by Leaflet.js
* Arabic RTL cinematic storytelling interface
* Two historical locations in Al-Kharj (Saudi Arabia)
* Smooth demo-based journey simulation
* Badge & achievement system for exploration
* LocalStorage-based progress tracking
* Collapsible sidebar UI with toggle control
* Fully responsive design (mobile & desktop)

⸻

📍 Locations Included

* 🏰 قصر الملك عبدالعزيز – الخرج

🎯 Core Concept

GeoRawi turns geographical exploration into a storytelling experience, where movement through space triggers historical narratives and rewards users with achievements for discovery.

With GeoRawi, you don’t go looking for places—you experience them before you even think to search. As you move through the city, stories find you, turning exploration into a guided journey of discovery.
